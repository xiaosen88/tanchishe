# Cyber Snake Game

一个赛博朋克风格的贪吃蛇游戏

## 特点

- 🎨 赛博朋克霓虹风格视觉设计
- 🎮 三种难度选择（简单/普通/困难）
- ✨ 粒子爆炸特效
- 🎵 音效系统（可开关）
- 🍎 特殊食物系统（双倍分数、加速效果）
- 📊 分数和等级系统
- 💾 最高分本地保存

## 控制方式

- 方向键或 WASD：移动蛇
- 空格键：暂停/继续
- R 键：重新开始
- ESC：暂停游戏

## 在线体验

[点击这里开始游戏](https://你的用户名.github.io/tanchishe/)

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/你的用户名/tanchishe.git

# 进入目录
cd tanchishe

# 启动本地服务器
npx http-server -p 8000

# 访问 http://localhost:8000
```

## 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6 模块)
- CSS3 动画
- Web Audio API

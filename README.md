# 贪吃蛇游戏

一个简约风格的贪吃蛇游戏

## 特点

- 🎨 简约清新的视觉设计
- 🎮 三种难度选择（简单/普通/困难）
- ✨ 粒子爆炸特效
- 🎵 背景音乐和音效系统（可开关）
- 🍎 特殊食物系统（双倍分数、加速效果）
- 📊 分数和等级系统
- 🏆 排行榜系统（TOP 10）
- 📈 游戏统计数据
- 💾 本地数据保存

## 新增功能

### 排行榜系统
- 记录前10名最高分
- 显示分数、难度、等级和日期
- 自动保存到本地存储

### 统计数据
- 总游戏次数
- 总得分和平均分
- 吃掉的食物总数
- 最高等级
- 各难度最高分

### 背景音乐
- 游戏开始时自动播放
- 可通过音效按钮开关

## 控制方式

- 方向键或 WASD：移动蛇
- 空格键：暂停/继续
- R 键：重新开始
- ESC：暂停游戏
- 移动端：支持滑动控制和虚拟按键

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/你的用户名/tanchishe.git

# 进入目录
cd tanchishe

# 启动本地服务器
npx http-server -p 8000

# 访问 http://localhost:8000
```

## 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6 模块)
- CSS3 动画
- Web Audio API
- LocalStorage
